# 📥 TriAuto Downloads

> **Organisez vos téléchargements sans effort.**  
> Une extension Chrome moderne qui trie automatiquement vos fichiers par type.

---

## ✨ Fonctionnalités

- 🚀 **Tri Intelligent** : Tes fichiers vont direct dans les bons dossiers (Images, Vidéos, PDF...) dès la fin du téléchargement.
- 🎨 **Interface Premium** : Un design épuré pour tout contrôler en un clic.
- ⚙️ **Ultra Personnalisable** : Choisis quelles extensions trier et gère tes propres catégories.
- 📊 **Statistiques** : Garde un œil sur le nombre de fichiers organisés.
- 🔒 **100% Privé** : Toutes tes données restent sur ta machine. Pas de cloud, pas de pistage.

---

## 📂 Catégories Supportées

L'extension gère intelligemment plus de 50 formats :

- 🖼️ **Images** : .jpg, .png, .svg, .webp...
- 🎬 **Vidéos** : .mp4, .mkv, .mov...
- 📄 **Documents** : .pdf, .docx, .xlsx, .txt...
- 🎵 **Musique** : .mp3, .wav, .flac...
- 📦 **Archives** : .zip, .rar, .7z...
- 💻 **Code** : .html, .js, .py, .php...

---

## 🛠️ Installation Rapide

1. **Télécharge** ou clone ce dépôt.
2. Va sur `chrome://extensions/` dans ton navigateur.
3. Active le **Mode développeur** (en haut à droite).
4. Clique sur **Charger l'extension non empaquetée**.
5. Sélectionne le dossier du projet. **C'est prêt !** 🚀

---

## ⚡ CI/CD & Releases

Ce projet utilise **GitHub Actions** pour automatiser les tâches répétitives :

- **Build Automatique** : À chaque push sur `main`, l'extension est compressée en `.zip` et disponible en tant qu'artefact.
- **Auto-Release** : Lorsqu'un tag (ex: `v1.3.1`) est poussé, une Release GitHub est créée automatiquement avec l'archive de l'extension prête à être installée.
